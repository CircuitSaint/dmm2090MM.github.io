Humanoid Systems Pinball Systems Test Bench

20 August 2018

 As an introductory excercise to the class, created systems using human beings. Our first system included everyone from the entire class. We had one person signally when the system would turn on, a person spinning another person, a person removing and replacing a person's hat, a person stored the values for the input data, and a person adjusting one set of lights based on the amount of rotations and turning on/off another set of lights based on the position of the cap. As a system check, another person read the stored values and signaled if the stored values and light adjustments were not true.
