The Beginning

In our first class, we learned how to adjust our orientation, create new sketches, and set our default unit of measurement in Fusion 360. I found it extremely helpful to go over the basics considering we will be using this software for another class.

Constraints

This class was a continuation of learning basics. We designed one part, while working on creating constraints to make parametric parts. This class was challenging as it was a lot of steps and the methodology is different from another software I'm familar with.

Making Connections

We dove into the process of creating parametric designs. By learning this process in fusion 360, I was able to reference my experience from OpenSCAD. My true test of parametric design was beginning the base for a flip jig. I needed to ensure I could change the height for different sized projects and to ensure the tool nor the spindle would ram into the jig.
