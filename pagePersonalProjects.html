Humanoid Systems Pinball Systems Test Bench

20 August 2018

 As an introductory excercise to the class, created systems using human beings. Our first system included everyone from the entire class. We had one person signally when the system would turn on, a person spinning another person, a person removing and replacing a person's hat, a person stored the values for the input data, and a person adjusting one set of lights based on the amount of rotations and turning on/off another set of lights based on the position of the cap. As a system check, another person read the stored values and signaled if the stored values and light adjustments were not true.

 I jumped into the power supply project a little late, story regarding that will be posted to Learning & Education soon. I had some success with the mini-projects leading up to this project. In all honesty, some intention success, some not so intentional. The purpose of the project was to make a power supply that would plug into a 120V wall outlet and output 5V. The power supplies would be used for the rest of the semester in future projects. I jumped on board with a classmate that organized the components on the perf board. I took over soldering the components to the perf board, and he soldered the transformer to the scrap computer power cables. While soldering, we discussed designs for the casing and how the perf baord, transformer, fuses, would fit properly and connect.
